## 项目信息

- **路径**: `C:\Users\乏味\kaoyan-companion`
- **技术栈**: Next.js 16 + React + TypeScript + Tailwind CSS + lucide-react
- **启动命令**: `npm run dev`（端口 3000）

## 已有功能

1. 考研倒计时（可修改目标日期）
2. 今日任务列表（增删改查、进度条、localStorage 持久化）
3. 专注计时器（开始/结束，记录今日学习时长）
4. 每日打卡（连续打卡天数统计）
5. 每日一句励志语录
6. 数据重置功能

## 后续可扩展方向

- 学习数据统计与图表
- 定时提醒/通知
- 错题本/笔记
- 各科进度管理
- 每周复习计划
- 暗黑模式
- 白噪音/番茄钟
- 部署到 Vercel（用户已有 deploy-to-vercel skill）

## 为什么用这个技术栈

用户是心理学学生、编程零基础。Next.js 全栈一体，Tailwind 写样式直观易懂，localStorage 省去数据库配置，让用户能最快看到成果、建立信心。

**Why:** 用户明确提到考研拖延严重，需要"属于自己的软件"来陪伴和监督。快速出成果比技术完美更重要。

**How to apply:** 后续加功能时继续保持"先跑起来再优化"的原则，每加一个功能就让他立刻看到效果。
