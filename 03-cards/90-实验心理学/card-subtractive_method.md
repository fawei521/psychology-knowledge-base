# 减法法 / Subtractive Method

## 定义
减法法（subtractive method），又称唐德斯减法法或减数法，是通过比较两种不同复杂程度任务的反应时差异，来估计某一特定心理加工阶段所需时间的方法。

## 核心要点
- 提出者为荷兰生理学家 **Franciscus C. Donders**（1868）。
- 基本任务类型为 **ABC 反应时**：A 反应（简单反应时）、B 反应（选择反应时）、C 反应（辨别反应时）。
- 核心假设是 **纯插入假设（pure insertion）**：增加一个加工阶段不会影响其他阶段的持续时间。
- 通过相减可得：
  - C − A = 刺激辨别时间
  - B − C = 反应选择时间
  - B − A = 辨别 + 选择总时间
- 典型结果：**B 反应时 > C 反应时 > A 反应时**。

## 理论背景
- Donders 受 Helmholtz 测量神经传导速度启发，试图用时间指标量化内部心理过程。
- 该方法首次将“不可观察”的心理过程纳入自然科学测量范畴。
- 后来的 Sternberg 相加因素法、神经影像学的认知减法（cognitive subtraction）均受其影响。

## 经典实验
- **Donders 视觉反应时任务**：
  - A 反应：看到任何光即按键。
  - C 反应：看到红光按键，绿光不按键。
  - B 反应：红光按右键，绿光按左键。
- 结果发现更复杂任务反应时更长，为相减估计奠定基础。
- 现代应用：Stroop 任务中冲突条件与一致条件反应时之差可视为干扰效应时间。

## 评价与争议
- **优势**：逻辑清晰、历史悠久、教学与研究价值高；是神经影像学减法设计的直接来源。
- **局限与批评**：
  - **纯插入假设不成立**：增加任务难度可能改变整体加工策略，而非单纯插入一个阶段。
  - **串行加工假设**：未考虑并行加工或级联加工。
  - **定性定量双重影响**：Külpe（1909）指出增加任务既影响时间，也改变加工性质。
- **与相加因素法的区别**：减法法依赖任务差异的相减；相加因素法依赖实验因素交互作用模式判断阶段是否相同。

## 生活实例
- 驾驶员从看到红灯到踩刹车：简单反应时；若同时有行人则需要选择反应时。
- 打字时听到任意声音就按键 vs. 听到特定字母才按键，体现辨别与选择反应时的差异。

## 考研重点
- **常考题型**：名词解释“减法法/唐德斯减法法”；简答 ABC 反应时及其计算关系；分析纯插入假设的局限。
- **易混淆点**：
  - A/B/C 三种反应分别对应哪些加工过程。
  - 减法法只能用于可明确分离且假设串行插入的阶段。
- **记忆口诀**：
  - “A 简单 B 选择 C 辨别，C 减 A 是辨别，B 减 C 是选择。”
