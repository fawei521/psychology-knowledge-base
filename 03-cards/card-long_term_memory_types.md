# 长时记忆的分类 / Types of Long-Term Memory

## 定义
长时记忆是信息经过充分加工后形成的、容量巨大且保持时间长久的记忆系统。根据信息内容与意识可及性，可分为陈述性记忆与程序性记忆；陈述性记忆又可进一步分为情景记忆与语义记忆。

## 核心要点
- 要点1：**陈述性记忆（declarative memory）** = "知道是什么"，可被意识提取和语言表达，包括情景记忆和语义记忆。
- 要点2：**程序性记忆（procedural memory）** = "知道怎么做"，表现为技能、习惯和条件反射，通常难以用语言完整表达。
- 要点3：**情景记忆（episodic memory）** 存储个人经历的特定事件，具有时间-地点标记（如"去年生日我在哪里"）。
- 要点4：**语义记忆（semantic memory）** 存储一般性知识和概念（如"巴黎是法国首都"、"三角形有三条边"），去情境化。
- 要点5：陈述性记忆主要依赖内侧颞叶/海马；程序性记忆主要依赖基底神经节和小脑。

## 理论背景
- 提出者/流派：Cohen & Squire（1980）区分"knowing that"与"knowing how"；Tulving（1972, 1983）区分情景记忆与语义记忆；Squire 长期研究记忆系统的神经基础。
- 经典实验/研究：遗忘症患者 H.M. 术后无法形成新的情景/语义记忆，但仍可学会镜像追踪等程序性技能；这成为两类记忆分离的关键证据。
- 发展脉络：单一长时记忆观 → 陈述性/程序性二分 → 情景/语义亚型 → 与内隐/外显记忆框架的整合。

## 评价与争议
- 优势：分类框架得到大量神经心理学和脑成像研究支持，能解释不同脑区损伤导致的记忆障碍差异。
- 争议：情景记忆与语义记忆的界限并非绝对；自传体记忆可能同时包含情景和语义成分。程序性记忆内部（技能、启动、条件反射）的同质性也受到质疑。

## 生活实例
- 例1：骑自行车多年不会忘——程序性记忆。
- 例2：记得小学毕业典礼的细节——情景记忆。
- 例3：知道"水在标准大气压下100°C沸腾"——语义记忆。

## 考研重点
- 常考题型：简答/论述（长时记忆的分类及各自特点）、名词解释（陈述性/程序性/情景/语义记忆）。
- 易混淆点：陈述性 vs 程序性；情景 vs 语义；陈述性/程序性 vs 外显/内隐（前者按内容与意识分，后者按提取意识分，有重叠但不等同）。
- 记忆口诀："陈述是知道，程序是会做；情景带时空，语义是常识"
