# C群人格障碍 / Cluster C Personality Disorders

## 定义

C群人格障碍以**焦虑、恐惧和回避**为主要特征，包括回避型、依赖型和强迫型人格障碍。该群与焦虑障碍有密切关系，常表现为对社交、分离或失控的过度担忧。

## 核心要点

- **回避型人格障碍**：社交抑制、自卑感、对负面评价过度敏感，渴望社交但因恐惧而回避。
- **依赖型人格 disorders**：过度需要被照顾、顺从和依附行为、害怕分离、难以独立决策。
- **强迫型人格障碍**：过度追求完美、秩序、控制，僵化固执，影响效率和人际关系。
- C群整体特征：高焦虑、回避、依赖或控制行为。
- 强迫型人格障碍（OCPD）≠ 强迫症（OCD）：前者是人格特质模式，后者是焦虑障碍，以强迫思维和强迫行为为特征。

## 理论背景

- **提出者/流派**：DSM 分类系统；C群反映“焦虑/恐惧”维度。
- **发展脉络**：
  - 早期理论强调习得性焦虑、依恋问题和过度控制的家庭环境。
  - 当代研究关注 C群与焦虑障碍、抑郁障碍的共病机制。
  - 维度取向认为 C群反映高神经质与低外向性的极端组合。
- **关键假设**：C群障碍与焦虑敏感性、依恋不安全、早期控制型教养方式有关。

## 经典实验/研究

- **共病研究**：回避型人格障碍与社交焦虑障碍高度共病。
- **依恋研究**：依赖型人格障碍与不安全依恋（尤其是焦虑型依恋）相关。
- **认知行为研究**：OCPD 与完美主义、僵化的认知风格有关。

## 评价与争议

- **优势**：C群分类有助于识别以焦虑为核心的人格病理，指导认知行为治疗。
- **争议/反例**：
  - 回避型人格障碍与社交焦虑障碍的区分在临床和研究中存在争议。
  - 依赖型人格障碍的诊断标准被认为可能受性别偏见影响。
- **与相近概念的区别**：
  - OCPD vs OCD：OCPD 是自我协调的人格模式，OCD 是自我不协调的强迫症状。
  - 回避型人格障碍 vs 社交焦虑障碍：前者是更广泛的人格模式，后者是特定的焦虑障碍。

## 生活实例

- 回避型人格障碍者渴望交朋友，但因害怕被拒绝而拒绝所有聚会邀请。
- 强迫型人格障碍者花大量时间整理文件，导致项目进度严重滞后。

## 考研重点

- **常考题型**：名词解释“回避型人格障碍”“强迫型人格障碍”；简答题“C群人格障碍的特点”。
- **易混淆点**：OCPD 与 OCD 的核心区别；依赖型人格障碍与依赖行为正常范围的区分。
- **记忆口诀**：C 组焦虑又恐惧，回避依赖加强迫；OCPD 非 OCD，人格模式要记牢。
