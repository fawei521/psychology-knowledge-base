# 回避型人格障碍 / Avoidant Personality Disorder (AvPD)

## 定义
DSM-5 中 C 群人格障碍之一，以社交抑制、自我价值感低下和对负面评价的过度敏感为核心特征。患者渴望社交联系，但因害怕被拒绝、羞辱或批评而广泛回避人际情境。

## 核心要点
- **核心特征**：社交抑制、自卑、对拒绝/批评过度敏感。
- **DSM-5 诊断标准（满足 ≥4 项）**：
  1. 因害怕批评、反对或拒绝而回避需要人际接触的职业活动；
  2. 除非确定被喜欢，否则不愿与人打交道；
  3. 因害怕被羞辱或嘲笑而在亲密关系中表现拘谨；
  4. 专注于在社交情境中被批评或拒绝；
  5. 在新的人际情境中因感到能力不足而受抑制；
  6. 认为自己在社交上无能、个人没有吸引力或低人一等；
  7.  unusually 不愿意冒个人风险或参与可能令人尴尬的新活动。
- **与社交焦虑障碍的关系**：两者高度共病（约 36% 的 AvPD 共病 SAD），但 AvPD 是更广泛的人格模式，核心自我评价更低。
- **功能损害**：职业、学业、亲密关系常严重受限。
- **治疗**：CBT、图式治疗、社交技能训练、团体治疗为主，治疗需缓慢建立信任。

## 理论背景
- **分类归属**：DSM-III 起归入 C 群“焦虑/恐惧型”。
- **病因理论**：
  - **遗传与气质**：行为抑制、高神经质、害羞等气质具有遗传性；与焦虑障碍共享遗传风险。
  - **依恋理论**：不安全依恋（尤其焦虑-矛盾型）可能增加风险。
  - **早期经验**：童年期反复羞辱、拒绝、欺凌、忽视或情感虐待，导致内化“自己有缺陷”的信念。
  - **认知因素**：核心信念包括“我不够好”“别人会拒绝我”“如果我暴露自己，会被羞辱”。
- **发展脉络**：儿童期害羞、社交退缩；青春期和成年早期因社交失败经验强化回避模式。

## 经典实验/研究证据
- **共病研究**：Alden 等（2002）综述显示 AvPD 与 SAD 共病率高达 21–89%，但约 2/3 的 AvPD 不满足 SAD 诊断，说明 AvPD 不仅是“严重社交焦虑”。
- **气质研究**：行为抑制儿童在青春期和成年后出现社交焦虑/回避型人格病理的风险升高（Biederman 等，2001）。
- **治疗研究**：Emmelkamp 等（2006）的 RCT 显示聚焦社交技能的 CBT 对 AvPD 有效；图式治疗在人格障碍治疗中证据逐渐积累。

## 评价与争议
- **优势**：AvPD 概念强调了长期自我贬抑与广泛回避模式，区别于情境性社交焦虑。
- **争议/反例**：
  - 与 SAD 的区分在临床和研究中存在持续争论；
  - 部分研究者认为 AvPD 是 SAD 的严重形式，应合并；
  - 诊断可能受文化对害羞与退缩的接受度影响。
- **与相近概念区别**：
  - 社交焦虑障碍：对特定社交情境的恐惧，自我评价相对保持；
  - 分裂样型人格障碍：对社交关系本身缺乏兴趣，而非害怕；
  - 依赖型人格障碍：核心是被照顾和害怕分离，不一定是社交回避。

## 生活实例
- 一个人非常渴望交朋友，但每次收到聚会邀请都因“我会说错话、别人会觉得我很无聊”而拒绝，长期孤立。
- 一名员工因害怕在会议上发言出错，主动回避晋升机会，尽管工作能力很强。

## 考研重点
- **常考题型**：名词解释“回避型人格障碍”；简答 AvPD 与社交焦虑障碍的异同；比较 C 群人格障碍。
- **易混淆点**：AvPD 与 SAD 的核心鉴别在于 AvPD 是人格水平的广泛模式，自我评价更低；SAD 更可能是情境性的。
- **记忆口诀**：“渴望社交怕拒绝，自卑敏感避人际；AvPD 是人格病，SAD 更像情境惧。”
