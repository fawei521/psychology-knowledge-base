# 测验等值 / Test Equating

## 定义

测验等值（Test Equating）是指对测量同一心理品质的不同测验形式，通过统计方法建立分数转换关系，使不同形式下的分数具有可比性和可互换性的过程。

## 核心要点

- **基本条件**：
  - **同质性**：不同形式测量同一种心理品质；
  - **等信度**：各形式的信度基本相等；
  - **公平性**：考生参加任一形式，等值后结果应相同；
  - **可递推性**：若 X = Y，Y = Z，则 X = Z；
  - **对称性**：等值转换可双向进行。
- **等值设计**：
  - **单组设计**：同一组被试接受两种形式；
  - **随机等组设计**：从同一总体随机抽取两组，分别接受不同形式；
  - **锚测验-非等组设计**：两组分别接受不同形式，同时都做共同锚题。
- **等值方法**：
  - **线性等值**：标准分数相等即等值；
  - **等百分位等值**：百分等级相等即等值；
  - **IRT 等值**：基于项目反应理论参数进行等值。
- **与 linking 的区别**：等值要求测验形式同质、等信度，分数可互换；linking 是不同测验之间的分数关联，分数不一定可互换。

## 理论背景

- **提出者/流派**：测验等值技术伴随大规模标准化测验发展；Kolen 与 Brennan 的著作系统总结了等值理论与方法。
- **发展脉络**：传统线性等值与等百分位等值 → IRT 等值 → 核等值（kernel equating）→ 计算机自适应测验中的实时等值。
- **关键假设**：
  - 不同测验形式测量同一构念；
  - 形式间难度差异可通过统计方法校正；
  - 样本或锚题能够提供可靠的等值信息。

## 经典实验

- **SAT 等值研究**：College Board 通过随机等组设计和锚测验设计，在不同考试日期之间建立等值关系，确保录取决策公平。
- **PISA 跨轮次等值**：国际学生评估项目通过共同题和 IRT 等值技术，使不同年份的分数具有可比性。

## 评价与争议

- **优势**：使不同时间、不同形式的测验分数具有可比性，是高利害测验和纵向研究的必要技术。
- **争议/反例**：
  - 等值假设（同质性、等信度）在现实中难以完全满足；
  - 锚题泄露或功能差异会影响等值准确性；
  - 等值不能解决测验内容差异带来的公平性问题。
- **与相近概念的区别**：等值是在同质形式间建立可互换分数；linking 是在不同测验间建立关联；垂直量表（vertical scaling）是在不同年级水平间建立分数连续体。

## 生活实例

- **高考不同年份分数比较**：由于每年试卷难度不同，通过等值技术可将不同年份的分数调整到同一尺度上比较。
- **托福 iBT 与纸笔考试**：ETS 通过等值研究建立两种考试形式的分数对应关系。

## 考研重点

- **常考题型**：名词解释、简答题（等值基本条件、等值方法）、选择题（等值与 linking 区别）。
- **易混淆点**：
  - 等值要求测验同质、等信度，分数可互换；linking 不一定满足；
  - 线性等值基于标准分数相等，等百分位等值基于百分等级相等；
  - 单组设计易受练习效应影响，随机等组设计需大样本。
- **记忆口诀**："等值同质又等信，公平递推对称行；线性等百 IRT，锚题连接保公平。"
