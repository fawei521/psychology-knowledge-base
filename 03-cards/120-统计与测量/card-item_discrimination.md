# 项目区分度 / Item Discrimination

## 定义

项目区分度（Item Discrimination）是指测验题目区分高能力被试与低能力被试的能力，也称鉴别力。区分度高的题目应使高分组通过率显著高于低分组。

## 核心要点

- **鉴别指数法（D 法）**：将总分排序后取上下各 27% 为高分组和低分组，计算 $D = P_H - P_L$。
  - D ≥ 0.40：很好
  - 0.30–0.39：良好，修改后更佳
  - 0.20–0.29：尚可，需修改
  - D < 0.20：较差，考虑删除
- **相关法**：
  - **点二列相关**：二分题目与连续总分的相关；
  - **二列相关**：连续变量人为二分时使用；
  - **φ 相关**：两个变量均为二分时使用。
- **项目-总分相关（rpbis）**：通常要求 rpbis ≥ 0.20，低于 0 或接近 0 表示题目质量差或答案有误。
- **与难度的关系**：难度在 0.50 左右时，区分度理论上最大；难度极端时区分度受限。
- **与信效度的关系**：高区分度题目通常能提升测验内部一致性和效标效度。

## 理论背景

- **提出者/流派**：Kelley (1939) 系统提出上下 27% 分组法；项目-总分相关由经典测量理论发展而来。
- **发展脉络**：经验性题目筛选 → 鉴别指数和相关法 → IRT 中的 a 参数（区分度参数）→ 现代测验中的多维项目分析。
- **关键假设**：
  - 总分能较好代表被试能力；
  - 样本能力分布近似正态；
  - 题目计分方式适合所选统计量。

## 经典实验

- **Kelley (1939)**：证明在正态分布下，上下 27% 分组法能在计算简便性和区分效果之间取得较好平衡。
- **Ebel (1965) 项目分析研究**：系统比较不同区分度指标（D 指数、点二列相关、phi 相关）在不同样本量和题目难度下的表现，为教育测验提供实践指南。

## 评价与争议

- **优势**：区分度是判断题目质量最直接的指标之一，高区分度题目有助于拉开被试差距。
- **争议/反例**：
  - 小样本时 27% 分组法不稳定，可能高估或低估区分度；
  - 若总分本身不可靠，以此计算的区分度也会失真；
  - 某些题目（如测谎题、态度题）可能预期与总分低相关，不能简单按区分度删除。
- **与相近概念的区别**：难度描述题目通过率；区分度描述题目对能力的区分能力；一道题可以很难但区分度很低（大家都不会）。

## 生活实例

- **驾照考试**：一道题如果高分考生和低分考生答对率差不多，说明它不能区分考生水平，可能需要替换。
- **招聘测评**：区分度高的题目能帮助公司识别能力强的应聘者；区分度低的题目增加测试长度却不提高选拔效果。

## 考研重点

- **常考题型**：计算题（D = PH − PL）、选择题（区分度评价标准）、简答题（区分度与难度的关系）。
- **易混淆点**：
  - 区分度可为负，表示题目质量差或答案设置错误；
  - 27% 分组法中的 27% 是经验值，非绝对标准；
  - 难度极端时，区分度最大值受限。
- **记忆口诀**："D 值等于高减低，零点四是优秀；负值出现要警惕，答案可能有误。"
