# 标准误 / Standard Error

## 定义

标准误（Standard Error, SE）是样本统计量抽样分布的标准差，用于衡量样本统计量（如样本均值）因随机抽样而产生的变异程度，即抽样误差的大小。

## 核心要点

- **样本均值的标准误公式**：$SE_{\bar{X}} = \frac{\sigma}{\sqrt{n}}$，其中 σ 为总体标准差，n 为样本容量。若 σ 未知，用样本标准差 s 估计：$SE_{\bar{X}} = \frac{s}{\sqrt{n}}$。
- **反映精确度而非离散度**：标准误越小，说明样本统计量对总体参数的估计越精确。
- **与样本量的关系**：标准误与样本量的平方根成反比；样本量扩大 4 倍，标准误减半。
- **与标准差的区别**：标准差描述原始数据的离散程度；标准误描述样本统计量的抽样变异。
- **应用**：标准误是构建置信区间、计算检验统计量（如 z、t）和确定误差范围的核心分量。

## 理论背景

- **提出者/流派**：标准误的概念与抽样分布理论同步发展，由 Fisher、Gosset 等人在 20 世纪初的形式化推断统计中确立。
- **发展脉络**：标准差（描述个体变异）→ 标准误（描述统计量变异）→ 标准误在 t 检验、方差分析、回归分析中的推广 → 稳健标准误（robust SE）处理异方差和聚类相关。
- **关键假设**：
  - 观测相互独立；
  - 样本来自同一总体；
  - 总体标准差 σ 已知（用 z）或未知（用 s 与 t 分布）。

## 经典实验

- **Student's t 检验的诞生**：Gosset（1908）在 Guinness 啤酒厂工作时发现，当总体标准差未知且样本量较小时，用样本标准差 s 估计的标准误会导致检验统计量不再服从正态分布，从而提出 t 分布，纠正了小样本下标准误估计的偏差。
- **标准误在元分析中的应用**：在元分析中，每个研究的效应量标准误用于计算权重，标准误越小的研究权重越大，从而得到更精确的综合估计。

## 评价与争议

- **优势**：标准误将样本量信息纳入误差估计，使研究者能够区分个体差异与抽样不确定性。
- **争议/反例**：
  - 当数据存在自相关、聚类结构或异方差时，常规标准误会低估真实抽样变异，需要使用稳健标准误或分层模型；
  - 样本标准差 s 本身也是估计量，小样本时估计的标准误不够稳定。
- **与相近概念的区别**：标准差（SD）描述数据本身；标准误（SE）描述统计量的抽样波动。报告时常写 "M = 12.3, SD = 4.1, SE = 0.82"。

## 生活实例

- **调查误差范围**：媒体称 "支持率 52%，误差 ±3%"，这个 ±3% 通常是 95% 置信区间的一半，由样本比例的标准误乘以 1.96 得到。
- **考试平均分**：两个班级平均分相同，但一个班 100 人、一个班 25 人。100 人班平均分的标准误更小，其平均分更可靠。

## 考研重点

- **常考题型**：计算题（已知 σ 或 s、n，求标准误）；选择题（区分 SD 与 SE、样本量对 SE 的影响）。
- **易混淆点**：
  - 标准误随样本量增加而减小，标准差不必然随样本量增加而减小；
  - 总体标准差已知时用 σ/√n 与 z 分布；未知时用 s/√n 与 t 分布。
- **记忆口诀**："标准差看个体，标准误看均值；根号 n 在分母，n 大误差低。"
