# 外显记忆 / Explicit Memory

## 定义

外显记忆（explicit memory）是指个体在有意识状态下主动提取过去经验的记忆形式。它包括情景记忆和语义记忆，是长时记忆中最常被意识和语言表达的部分。

## 核心要点

- **有意识提取**：个体能够意识到自己在回忆，并可用语言报告记忆内容。
- **主要类型**：
  - **情景记忆（episodic memory）**：对个人经历的具体事件及其发生时间、地点的记忆。
  - **语义记忆（semantic memory）**：对一般知识、概念、事实和语词意义的记忆，去情境化。
- **测验方式**：自由回忆、线索回忆、再认等。
- **神经基础**：主要依赖内侧颞叶（尤其是海马）、间脑和前额叶皮层；海马损伤会导致严重的外显记忆障碍。
- **遗忘特点**：外显记忆容易受时间、干扰、情绪、提取线索等因素影响。

## 理论背景

- **提出者/流派**：Tulving（1972, 1983）区分情景记忆与语义记忆；Graf & Schacter（1985）系统提出外显/内隐记忆区分。
- **发展脉络**：从单一长时记忆观 → 陈述性/程序性二分 → 情景/语义亚型 → 外显/内隐提取区分 → 与神经影像和临床研究的整合。
- **关键假设**：外显记忆依赖有意识提取过程，其信息内容和提取情境可被主观报告。

## 经典实验

- **研究名称/研究者/年份**：Tulving & Pearlstone（1966）可获得性 vs 可提取性实验。
- **实验设计/关键操作**：
  - 被试学习 categorized 单词列表，随后进行自由回忆或提供类别线索的回忆。
- **主要发现及其与概念的关系**：
  - 提供类别线索显著提高了回忆成绩。
  - 说明信息在长时记忆中“可获得”（available）但不一定“可提取”（accessible），外显记忆提取依赖有效线索。

## 评价与争议

- **优势**：外显记忆框架能解释日常有意识回忆、学习和考试表现；与神经心理学临床证据高度吻合。
- **争议/反例**：
  - 情景记忆与语义记忆的界限并非绝对，自传体记忆常同时包含两者。
  - 外显记忆测验也可能被内隐过程污染。
- **与相近理论的区别**：
  - 与内隐记忆：外显记忆需要意识提取，内隐记忆自动影响行为。
  - 与陈述性记忆：陈述性记忆按内容分，外显记忆按提取意识分，两者高度重叠。

## 生活实例

- 例1：考试时主动回忆知识点——外显记忆提取。
- 例2：记得去年生日在哪里过、和谁一起——情景记忆。
- 例3：知道“巴黎是法国首都”——语义记忆。

## 考研重点

- **常考题型**：名词解释（外显记忆/情景记忆/语义记忆）、简答（外显记忆与内隐记忆的区别）、论述（长时记忆的分类及提取方式）。
- **易混淆点**：
  - 外显/内隐 vs 陈述性/程序性：划分标准不同。
  - 情景记忆 vs 语义记忆：前者有个人时间-地点标记，后者去情境化。
- **记忆口诀**：「外显记忆意识提，情景语义两兄弟；回忆再认常用测，海马受损大问题。」
