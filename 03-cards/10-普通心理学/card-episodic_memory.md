# 情景记忆 / Episodic Memory

## 定义

情景记忆（episodic memory）是指个体对亲身经历的具体事件及其发生时间、地点、情境和情绪的记忆。它使人能够“心理时间旅行”，重新体验过去的经历。

## 核心要点

- **个人性与情境性**：情景记忆存储的是“我何时何地经历了什么”，具有明确的时间-地点标记。
- **主观体验**：回忆情景记忆时常伴随生动形象、情绪色彩和自我参与感。
- **与语义记忆的关系**：
  - 情景记忆是“知其然”（知道事件），语义记忆是“知其所以然”（知道事实）。
  - 多次重复的情景经验可抽象为语义知识（如多次去故宫后知道“故宫在北京”）。
- **神经基础**：主要依赖海马和内侧颞叶，前额叶皮层参与提取和组织。
- **发展变化**：儿童早期情景记忆能力较弱，随海马成熟和自我意识发展而增强；老年期情景记忆常较早衰退。

## 理论背景

- **提出者/流派**：Endel Tulving（1972）首次区分情景记忆与语义记忆；Tulving（1983, 2002）进一步发展“心理时间旅行”和自我意识理论。
- **发展脉络**：从单一长时记忆观 → Tulving 的情景/语义二分 → 自传体记忆研究 → 神经影像对海马作用的确认。
- **关键假设**：情景记忆不仅是对事实的存储，更是对“自我在特定时空中的经验”的重构。

## 经典实验

- **研究名称/研究者/年份**：Tulving（1972, 1983）理论建构；Vargha-Khadem 等（1997）海马损伤儿童研究。
- **实验设计/关键操作**：
  - Vargha-Khadem 等追踪早期海马损伤儿童，发现他们能获得语义知识，但无法形成新的情景记忆。
- **主要发现及其与概念的关系**：
  - 海马损伤严重影响情景记忆，但语义记忆可部分保留。
  - 支持情景记忆与语义记忆在神经基础上的部分分离。

## 评价与争议

- **优势**：情景记忆概念解释了个人历史感、自我连续性和自传体经验的认知基础。
- **争议/反例**：
  - 情景记忆与语义记忆的边界并非绝对；许多记忆同时包含两者。
  - 情景记忆的准确性易受重构、暗示和情绪影响。
- **与相近理论的区别**：
  - 与语义记忆：情景记忆带有时空和自我标记，语义记忆去情境化。
  - 与闪光灯记忆：闪光灯记忆是情景记忆的一种特殊、情绪唤醒强烈的子类型。

## 生活实例

- 例1：记得小时候第一次坐火车的情景——情景记忆。
- 例2：能回忆起昨天午餐吃了什么、和谁一起——情景记忆。
- 例3：多次参观博物馆后，知道“卢浮宫在巴黎”——语义记忆从情景经验中抽象而来。

## 考研重点

- **常考题型**：名词解释（情景记忆/心理时间旅行）、简答（情景记忆与语义记忆的区别与联系）、论述（Tulving 的长时记忆分类）。
- **易混淆点**：
  - 情景记忆 vs 语义记忆：前者有个人经历标记，后者是通用知识。
  - 情景记忆 vs 自传体记忆：自传体记忆更宽泛，情景记忆是其核心成分。
- **记忆口诀**：「情景记忆带时空，自我经历能重逢；多次抽象变语义，海马受伤最吃亏。」
