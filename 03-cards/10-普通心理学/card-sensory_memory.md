# 感觉记忆 / Sensory Memory

## 定义

感觉记忆（sensory memory）是指外界刺激停止作用后，感觉信息在感觉通道中保持极短时间的记忆系统。它是记忆三级模型的最前端，容量大、保持时间短，信息经注意筛选后进入短时记忆。

## 核心要点

- **图像记忆（iconic memory）**：视觉感觉记忆，保持时间约 0.25–1 秒，容量较大。
- **声像记忆（echoic memory）**：听觉感觉记忆，保持时间约 2–4 秒，长于图像记忆。
- **容量大、保持极短**：感觉记忆能同时保存大量感觉信息，但若不立即被注意加工，会迅速衰退。
- **注意是筛选门控**：信息从感觉记忆进入短时记忆依赖于选择性注意；未被注意的信息通常无法进一步加工。
- **功能**：保持刺激的连续性，使我们在视觉或听觉上形成稳定的知觉经验。

## 理论背景

- **提出者/流派**：George Sperling（1960）通过部分报告法系统研究图像记忆；Neisser（1967）提出 echoic memory 概念。
- **发展脉络**：早期对瞬时记忆的兴趣源于 Wundt 的知觉研究 → Sperling 部分报告法量化图像记忆容量 → 与信息加工模型结合，成为 Atkinson-Shiffrin 三级记忆模型的输入端。
- **关键假设**：感觉记忆是自动化的、前注意的（pre-attentive）存储系统；其衰退速度取决于感觉通道的神经生理特性。

## 经典实验

- **研究名称/研究者/年份**：Sperling（1960）部分报告法实验。
- **实验设计/关键操作**：
  - 快速呈现 3×4 或 3×3 字母矩阵约 50 毫秒。
  - 全报告条件要求被试报告所有看到的字母；部分报告条件在刺激消失后立即呈现高/中/低音调提示，要求被试报告对应行的字母。
- **主要发现及其与概念的关系**：
  - 全报告法仅能报告约 4–5 个字母，部分报告法可报告提示行中约 3 个字母，推知图像记忆容量约为 9 个字母以上。
  - 部分报告成绩随提示延迟迅速下降，说明图像记忆保持时间极短。

## 评价与争议

- **优势**：部分报告法克服了全报告法中记忆衰退和输出干扰的问题，揭示了感觉记忆的大容量特性。
- **争议/反例**：
  - 感觉记忆是否应被视为“记忆”还是“知觉后效”存在讨论。
  - 不同感觉通道的感觉记忆特性差异较大，统一模型可能过于简化。
- **与相近理论的区别**：
  - 与短时记忆：感觉记忆容量大、保持极短、前注意；短时记忆容量有限、需复述保持、可被意识访问。
  -与注意：注意从感觉记忆中选择信息进入短时记忆，但感觉记忆本身不依赖注意。

## 生活实例

- 例1：看动画片时，连续静止画面因图像记忆的短暂保持而被知觉为流畅运动。
- 例2：在嘈杂聚会中，别人喊你名字时你仍能回头——声像记忆保留了刚才的听觉信息。
- 例3：抬头看一眼车牌，眨眼间就忘了——信息未被注意转入短时记忆。

## 考研重点

- **常考题型**：名词解释（感觉记忆/图像记忆/声像记忆）、简答（Sperling 部分报告法及其结论）。
- **易混淆点**：
  - 图像记忆 vs 声像记忆：前者视觉、约 0.25–1 秒；后者听觉、约 2–4 秒。
  - 感觉记忆 vs 短时记忆：前者保持极短、容量大、不被意识访问。
- **记忆口诀**：「容量大、时间短，感觉记忆像快照；注意选中进短时，Sperling 部分报告妙。」
