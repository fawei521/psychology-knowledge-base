# 结构-预期-自我实现循环

## 一句话定义
社会文化脚本内化为个人的关系预期图式，这些预期通过选择性感知、防御行为和试探反应，制造出符合预期的结果，从而加固原有结构。

## 循环模型

```
社会文化脚本
（男强女弱、供养者/被供养者、测试者/被测试者）
        ↓
形成亲密关系"预期图式"
（我会被抛弃 / 我会被利用 / 我不能失信）
        ↓
进入关系前就开始扫描危险信号
        ↓
把中性言行解读为危险信号
        ↓
用防御或试探回应
        ↓
对方反应发生变化（困惑、退缩、反击）
        ↓
反应"证明"了预期
        ↓
结构被加固，下次触发更快
```

## 学术背景

### 1. 拒绝敏感性与自我实现预言
Downey 等人（1998）的经典研究发现，高拒绝敏感性的人在亲密关系中会表现出更易引发拒绝的行为，从而形成"害怕被拒绝 → 行为引发拒绝 → 信念被确认"的循环。

### 2. 积极错觉的自我实现
Murray 等人（1996）则发现另一端：对伴侣的理想化感知也能成为自我实现预言，提升关系满意度。这说明循环既可以是负面的，也可以是正面的。

### 3. 依恋图式
依恋理论认为，早期关系经验形成"内部工作模式"，成为人们解释伴侣行为的模板。焦虑型依恋者更容易把中性信号解读为威胁。

### 4. 循环因果（Circular Causality）
家庭系统治疗反对"谁先说/谁先错"的线性因果，而强调循环因果：A 的行为是对 B 上一步行为的回应，B 的行为又是对 A 的回应。冲突一旦启动，每个人都是维持者。

### 5. 投射性认同
不仅自己把对方想象成某种角色，还通过言行把对方推向那个角色。例如：反复测试对方"够不够坚定"，最终把对方逼到退缩，从而"验证"了"他果然不够坚定"。

## 关键节点

| 节点 | 表现 | 心理机制 |
|---|---|---|
| 脚本内化的来源 | 家庭、媒体、同辈、网络 | 社会学习 |
| 触发点 | 进入亲密关系或关系压力事件 | 依恋系统激活 |
| 解读偏误 | 把中性/模糊信息读成威胁 | 确认偏误 |
| 行为反应 | 试探、防御、控制、退缩 | 依恋策略 |
| 对方反应 | 困惑、疲惫、反击、撤退 | 关系反馈 |
| 信念加固 | "看吧，果然如此" | 自我实现预言完成 |

## 如何打断循环

1. **觉察脚本**：识别"这是社会脚本在说话，不是事实"。
2. **延迟反应**：在情绪高峰时不做重大回应。
3. **寻找反例**：主动回忆伴侣不符合预期的时刻。
4. **元沟通**：和伴侣一起把结构当成共同敌人。
