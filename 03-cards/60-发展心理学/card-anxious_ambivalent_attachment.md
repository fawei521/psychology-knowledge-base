# 焦虑-矛盾型依恋 / Anxious-Ambivalent Attachment

## 定义

焦虑-矛盾型依恋是指婴儿在面对不一致、不可预测的照顾者回应时，形成对亲近高度渴望但又难以被安抚、在重聚时对照顾者既寻求接触又表现愤怒的依恋模式。

## 核心要点

- 在 Ainsworth 的陌生情境分类中，焦虑-矛盾型对应 Type C，也称"抗拒型"（resistant），约占 10–15%。
- 形成条件：照顾者回应不一致——有时敏感，有时忽视或拒绝，使婴儿无法形成稳定预期。
- 行为特征：分离时极度痛苦；重聚时既寻求接触又抗拒、踢打、难以平静；对探索兴趣低，持续警觉照顾者行踪。
- 内部工作模型："他人不可靠，我必须时刻警惕并强烈表达需求，才能得到回应。"
- 在成人依恋访谈中对应"沉溺型"（Preoccupied, E），表现为被童年记忆淹没、叙述愤怒或被动。
- 成人浪漫关系中常表现为"焦虑型依恋"：害怕被抛弃、寻求反复确认、情绪起伏大。

## 理论背景

- 提出者/流派：Mary Ainsworth（1978）描述；Cassidy & Berlin（1994）进一步研究其前因与后果。
- 发展脉络：Bowlby 的依恋理论 → Ainsworth 的陌生情境 Type C → Main 的成人依恋访谈分类 → Hazan & Shaver 的成人浪漫关系研究。
- 关键假设：不一致的照顾导致婴儿形成"放大"依恋信号的倾向，以提高获得回应的概率。

## 经典实验

- Ainsworth 的家庭观察（Ainsworth et al., 1978）：焦虑-矛盾型婴儿的母亲往往在婴儿不痛苦时侵入性地互动，在真正痛苦时回应迟钝或不当。
- Cassidy & Berlin（1994）综述：系统梳理了不一致照顾与焦虑-矛盾型依恋的关系，以及其后续的情绪调节困难。

## 评价与争议

- 优势：揭示了"不一致回应"对婴儿的影响，解释了矛盾行为的适应性意义。
- 争议/反例：焦虑-矛盾型行为可能与婴儿气质（如高反应强度）交互作用；类型标签可能掩盖连续维度的个体差异。
- 与相近理论的区别：与"难养型气质"相比，焦虑-矛盾型依恋强调关系历史；与成人焦虑型依恋相比，婴儿 Type C 是行为观察分类。

## 生活实例

- 焦虑-矛盾型婴儿在母亲短暂离开后大哭，母亲回来想抱他时，他却扭开身体或打母亲，既想亲近又表现愤怒。
- 焦虑-矛盾型成人可能会频繁发消息确认伴侣位置，伴侣回复稍晚就感到被忽视，冲突时情绪激烈。

## 考研重点

- 常考题型：名词解释（焦虑-矛盾型依恋）、简答题（焦虑-矛盾型依恋的行为特征与形成条件）。
- 易混淆点："矛盾"指既寻求亲近又抗拒亲近；与回避型的"压抑亲近"不同。
- 记忆口诀：焦虑矛盾难安抚，分离痛苦重聚怒；他人不可靠放大声，寻求亲近又抗拒。
