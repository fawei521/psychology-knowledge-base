# 社会事件心理认知分类手册（精简版）

> 本手册只做一件事：**看事件 → 判断主类型 → 推荐可调用的心理学理论**。
> 不负责组织概念卡片（概念归 03-cards 学科体系管），不替代具体分析。

## 一、八类总览

| 类型 | 核心问题 | 主导情绪 | 关键词 | 应调用的理论 |
|------|---------|---------|--------|------------|
| **威胁-防御** | 这会伤害我吗？ | 恐惧、焦虑 | 害怕、安全、风险、怎么办 | 恐惧管理理论、保护动机理论、风险感知 |
| **不公-愤怒** | 这公平吗？ | 愤怒、义愤 | 凭什么、不公平、双标、严惩 | 相对剥夺理论、公正世界信念、系统合理化 |
| **道德-厌恶** | 这突破底线了吗？ | 厌恶、鄙视 | 恶心、畜生、底线、玷污 | 道德基础理论、洁净道德、道德恐慌 |
| **共情-利他** | 他受苦了，我能做什么？ | 同情、怜悯 | 心疼、看哭了、想帮他、捐款 | 共情-利他假说、旁观者效应、可识别受害者 |
| **认同-团结** | 这关乎我们是谁？ | 群体自豪/愤怒 | 我们XX人、丢人、团结、一致对外 | 社会认同理论、群际情绪、群体极化 |
| **失调-困惑** | 到底怎么回事？ | 不确定、困惑 | 不信、等反转、细思极恐、矛盾 | 认知失调、归因理论、认知闭合需求 |
| **权力-支配** | 谁在压迫谁？ | 无力感、屈辱 | 压迫、底层、权贵、反抗、跪久了 | 社会支配理论、无权感、系统合理化 |
| **新奇-探索** | 这有趣/新奇吗？ | 好奇、惊讶 | 居然、吃瓜、活久见、求科普 | 信息缺口理论、社交货币、模因理论 |

## 二、分类决策树

```
事件进入认知加工
    │
    ├─→ 触发防御性回避？（恐惧、焦虑）
    │       └─→ 威胁-防御
    │
    ├─→ 触发接近性参与？
    │       ├─→ 惩罚/纠正动机 + 愤怒 → 不公-愤怒
    │       ├─→ 惩罚/纠正动机 + 恶心 → 道德-厌恶
    │       ├─→ 帮助/减轻痛苦 → 共情-利他
    │       ├─→ 维护群体/集体行动 → 认同-团结
    │       ├─→ 反抗控制/争取自主 → 权力-支配
    │       └─→ 探索/分享/娱乐 → 新奇-探索
    │
    └─→ 无法判断/信息矛盾 → 失调-困惑
```

## 三、混合事件规则

1. **主类型** = 事件本身直接激活的主导情绪-动机
2. **副类型** = 舆论发酵后叠加的次要激活
3. **标注格式**: `主类型 / 副类型1 / 副类型2`

**示例**:
- 唐山打人：`道德-厌恶 / 威胁-防御 / 不公-愤怒`
- 碳水面事件：`认同-团结 / 权力-支配`
- 彩礼事件：`认同-团结 / 不公-愤怒 / 失调-困惑`

## 四、事件 YAML 标注规范

在 `05-observations/` 下的事件文件中，添加：

```yaml
event_type:
  primary: "认同-团结"
  secondary: ["不公-愤怒", "失调-困惑"]
```

## 五、理论速查（按类型）

| 类型 | 核心理论（3-5个） |
|------|-----------------|
| 威胁-防御 | 恐惧管理理论(Greenberg et al.)、保护动机理论(Rogers)、风险感知(Slovic) |
| 不公-愤怒 | 相对剥夺(Runciman, 1966)、公正世界信念(Lerner, 1980)、第三方惩罚(Fehr, 2004) |
| 道德-厌恶 | 道德基础理论(Haidt, 2007)、洁净道德(Rozin, 2000)、道德恐慌(Cohen) |
| 共情-利他 | 共情-利他假说(Batson, 1991)、旁观者效应(Latane & Darley, 1968)、可识别受害者(Small, 2003) |
| 认同-团结 | 社会认同(Tajfel, 1979)、自我分类(Turner, 1987)、群际情绪(Mackie, 2000)、群体极化 |
| 失调-困惑 | 认知失调(Festinger, 1957)、归因理论(Weiner, 1985)、认知闭合(Kruglanski, 1990) |
| 权力-支配 | 社会支配(Sidanius, 1999)、无权感、系统合理化(Jost, 1994) |
| 新奇-探索 | 信息缺口(Loewenstein, 1994)、社交货币(Berger, 2011)、模因理论(Dawkins) |

## 六、每个类型的简要判断提示

### 威胁-防御
问：评论区的人是在找防护方法，还是在攻击责任人？→ 前者指向威胁-防御。

### 不公-愤怒
问：愤怒有明确的责任方吗？→ 有明确指向 = 不公-愤怒（区别于泛化的道德厌恶）。

### 道德-厌恶
问：高频词是"恶心/底线/不是人"还是"不公平/凭什么"？→ 前者指向道德-厌恶。

### 共情-利他
问：人们是在讨论"受害者多可怜"还是在讨论"谁对谁错"？→ 前者指向共情-利他。

### 认同-团结
问：讨论中"我们/他们"的边界是否在强化？→ 是 = 认同-团结被激活。

### 失调-困惑
问：评论区是否以"等反转/不信/到底怎么回事"为主？→ 是 = 失调-困惑。

### 权力-支配
问：叙事框架是"强欺弱"还是"对/错"？→ 前者指向权力-支配。

### 新奇-探索
问：转发量远大于深度评论量？→ 是 = 新奇-探索占主导。
