# 05-observations: 观察与分析层

> 存放你的个人经验、典型案例、热点时事心理分析。

## 子目录

- `personal-experiences/`：个人经历、行为模式、情绪反应记录
- `typical-cases/`：典型案例（可匿名化）
- `current-events/`：热点时事的心理学分析

## 文件命名

- `personal-experiences/px-{主题}-{日期}.md`
- `typical-cases/case-{主题}-{日期}.md`
- `current-events/event-{事件}-{日期}.md`

## YAML 字段规范

```yaml
---
observation_type: "personal_experience"  # personal_experience / typical_case / current_event
title: "标题"
event_date: "2026-06-12"
context: "事件发生的情境"
behavior: "观察到的行为或反应"
tags: ["tag1", "tag2"]
related_entities: ["big_five_personality", "attachment_theory"]
sources: ["summary-source-2024"]
---

## 分析
...

## 可提炼的经验
...
```
