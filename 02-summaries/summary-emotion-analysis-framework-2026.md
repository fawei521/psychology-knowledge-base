# Emotion Analysis Framework

## 研究问题

How can we infer emotional states from observable signals?

## 核心发现

1. Emotion should be analyzed through facial expression, vocal cues, body posture, verbal content, and context.
2. Establishing a baseline is essential before detecting changes.
3. Cultural norms shape emotional expression.

## 适用边界

- Emotion analysis is probabilistic, not deterministic.
- Microexpressions alone are not valid evidence.
