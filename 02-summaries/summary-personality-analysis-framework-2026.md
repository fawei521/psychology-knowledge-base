# Personality Analysis Framework

## 研究问题

How can we systematically analyze an individual's personality using psychological frameworks?

## 核心发现

1. Big Five (Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism) provides a robust trait description.
2. Attachment theory explains intimacy and conflict patterns.
3. MBTI has limited scientific validity but can describe communication preferences.
4. Analysis should combine observation, self-report, and third-party feedback.

## 适用边界

- Not for clinical diagnosis.
- Requires consent when analyzing others.
- Cultural differences affect expression.
