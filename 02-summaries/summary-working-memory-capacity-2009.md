# 工作记忆容量与个体差异

## 研究问题

工作记忆容量是否存在稳定限制？个体差异的来源是什么？

## 方法

综合多项行为实验和元分析。

## 核心发现

1. 工作记忆容量平均约为 5±12个组块。
2. 个体间差异显著，与注意控制、流体智力相关。
3. 组块化策略可以扩大有效容量。

## 适用边界

- 主要来自西方样本
- 任务类型会影响容量估计
