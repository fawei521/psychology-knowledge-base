3
自动标签 — auto_tag.py
命令
python tools/auto_tag.py
预览模式，不修改文件
确认后执行
python tools/auto_tag.py --apply
实际写入标签
🃏 概念卡片
匹配逻辑
扫描 domain + tags + concept 字段
对照 13 学科关键词表
补全 #学科/xxx 标签
🔍 事件文件
匹配逻辑
扫描 YAML 字段 + 正文前 800 字
对照 8 种事件类型关键词表
≥2 个关键词命中才确认
补全 #事件类型/xxx 标签
为什么概念卡只看 YAML，事件要扫正文？
概念卡的 domain 字段已明确学科，不需要扫正文。事件文件的 YAML 通常太简略（只有标题），需要读正文前几段才能判断类型。限制前 800 字是为了避开分析区的引用讨论。
4
同步数据库 — import_md.py
命令
python tools/import_md.py
增量同步
对比文件 mtime + hash
只处理 新增 / 修改 / 删除 的文件
不会每次全量重建

📦 kb.db（SQLite 数据库）

存储表：sources · snippets · entities · observations · relations · tags · vec_documents（向量）

数据库由 Markdown 文件生成，可随时删除重建——Markdown 是唯一真相源
