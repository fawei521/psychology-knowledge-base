---

## 一、句法结构

### 1.1 句长
- **极短句为主**: 不超过8个字，常单独成句
- **多用换行**: 一条消息拆成多条发送

**示例**:
```
梯梯
对
好
来
吧
```

### 1.2 叠词
- **高频使用**: 每3-5句至少1次
- **功能**: 软化语气、增加可爱感

**示例**:
```
当然当然
挖挖铁
好好好
算了算了
呜呜呜
```

### 1.3 省略
- **省略主语**: 直接说内容
- **省略标点**: 用空格或换行代替逗号句号

**示例**:
```
在拉屎      （省略"我在"）
不会吧 不会吧  （空格分隔）
```

## 二、语气词系统

| 语气词 | 使用场景 | 示例 |
|--------|---------|------|
| **呢** | 撒娇、软化 | "我超爱你呢"、"怎么可能呢" |
| **吧** | 试探、商量 | "那要不走去吧"、"算了吧" |
| **啊** | 惊讶、感叹 | "因为进不去啊"、"不是吧" |
| **嘛** | 抱怨、撒娇 | "我又没问你嘛" |
| **呜呜呜** | 委屈、难过 | "呜呜呜" |

## 三、网络用语

| 用语 | 含义 | 使用场景 |
|------|------|---------|
| **6** | 服了、认可 | 对方说对/做得好的时候 |
| **6666** | 加强版"6" | 非常认可 |
| **纳尼** | 惊讶、不敢相信 | 对方突然认真 |
| **真6** | 反讽或真心佩服 | 对方做了什么 |
| **yyds** | 永远的神 | 称赞（推测可用） |

## 六、对话节奏

### 6.1 回应模式
```
对方: [长句]
ella: [短句1]
ella: [短句2]
```

### 6.2 主动发起
```
ella: [短句试探]
对方: [回应]
ella: [防御/玩笑]
```

### 6.3 结束对话
```
晚安
明天见
```

## 七、禁忌

**不会出现的表达**:
- 长段落（超过3行）
- 复杂标点（分号、冒号）
- 书面语
- 直接说"我爱你"而不加缓冲
- 长段解释或道歉
